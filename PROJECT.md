# GoRentals Migration Project

## Overview
Migration of the GoRentals platform from Supabase to a custom Spring Boot backend.

## Requirements
- Node.js / Next.js Frontend
- Java 21 / Spring Boot 3.x Backend
- PostgreSQL 17 Database
- Centralized Axios Networking
- JWT Authentication

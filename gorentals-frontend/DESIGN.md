# Design System Document: GoRentals Premium Marketplace

## 1. Core Colors
- **Primary:** `#4F46E5` (Indigo) - Used for all active, verified, and success states. NO green.
- **Secondary:** `#1E3A8A` (Deep Blue) - Used for primary brand elements and deep contrast.
- **Tertiary:** `#A54100` (Amber-Brown) - Used for accents and highlights.
- **Neutral:** `#777681` - Used for secondary text and subtle UI elements.

## 2. Appearance
- **Light Mode:** White (`#FFFFFF`) and Slate-50 (`#F8FAFC`) surfaces with soft shadows.
- **Dark Mode:** Deep Slate-900 (`#0F172A`) surfaces with high-contrast readable text.

## 3. Typography
- **Font Family:** Inter (All weights)
- **Scale:** Modular scale for headers and body text.

## 4. UI Logic
- **Success/Action:** Indigo (`#4F46E5`) instead of green.
- **Shadows:** Soft, ambient shadows for depth.
